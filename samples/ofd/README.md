# OFD Sample Registry

状态：OFD-L0 样例登记入口
最后更新：2026-05-01

本目录只登记公开、可测试、无敏感信息的 OFD 样例。真实用户文件、业务票据、带个人信息或签章私钥风险的文件不得进入仓库。

## 收录规则

- 样例必须公开可获取，或明确允许用于开源测试。
- 样例不得包含姓名、身份证号、手机号、地址、业务编号、合同编号、账户信息、签章私钥或其他敏感信息。
- 样例必须记录来源、授权、页面数、字体依赖、签章状态、附件状态和预期输出。
- 样例应优先提供对照 PDF/PNG 或官方查看器截图，用于后续视觉 diff。
- 如无法确认安全性或授权，样例不得提交。

## Capability 对应

- OFD-L0：容器、manifest、metadata、页面数量、资源索引。
- OFD-L1：页面树、文本对象、图片对象、附件列表到 DocumentModel。
- OFD-L2：本地 PNG/PDF 渲染基础。
- OFD-L3：字体、矢量、签章、页面定位高保真专项。
- OFD-L4：回归样例库、视觉 diff、chunked 等价和资源预算。
